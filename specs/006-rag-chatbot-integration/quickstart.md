# Quickstart: RAG Chatbot

This document provides instructions on how to run the RAG chatbot.

## Backend

1.  Navigate to the `backend` directory.
2.  Install dependencies: `npm install`
3.  Run the server: `npm start`

## Frontend

1.  Navigate to the root directory.
2.  Install dependencies: `npm install`
3.  Run the Docusaurus site: `npm run start`
4.  Open the site in your browser and navigate to the `/chatbot` page.
